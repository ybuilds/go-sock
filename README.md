**GO SOCK**

A simple app to facilitate chatting between multiple users
Written purely in Golang

**How to run?**
- Clone the repository
- Fire up two instances of terminal, one for server and one for client
- If you wish to use the websocket version of the server, create an approproate client app for that.
- Else you can run the given client app directly in your terminal
- Run "go run ." from both the terminals. (First the server, then the client. Server has to be UP for the client to connect)
- Enjoy chatting!
